RidePartner MP Driver Rules

1. Wallet aur Commission

Har safar poora hone par 10% commission automatic deduct hoga. ID active rakhne ke liye balance ₹0 se upar rakhein.

2. Cancellation Penalty

Accept karne ke baad ride cancel karne par ₹10 se ₹50 tak penalty lag sakti hai jo aapke wallet se kategi.

3. OTP aur Suraksha

Safar shuru karne se pehle passenger se OTP maangna zaroori hai. Bina OTP ke ride system mein valid nahi hogi.

4. Sahi Vyavhar

Passenger ke saath misbehave karne ya offline ride le jane par ID hamesha ke liye block kar di jayegi.
